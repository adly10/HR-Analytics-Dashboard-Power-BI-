# 📊 HR Analytics Dashboard (Power BI)

## 📌 Overview

This project presents an interactive **HR Analytics Dashboard** built using Power BI to analyze workforce trends, employee distribution, and key HR metrics.

The goal is to help HR teams make **data-driven decisions** regarding employee management, retention, and organizational structure.

---

## 📁 Dataset

* Source: Internal HR dataset (CSV format)
* Records include:

  * Employee demographics
  * Salary information
  * Department data
  * Employment status
  * Education level

---

## 📊 Key Metrics

* 👥 Total Employees: **3520**
* 💰 Median Salary: **95K**
* 🏥 Sick Days: **23K**
* 📉 Balance Days: **44K**
* 🎂 Average Age: **44.33**
* 🏢 Departments: **15**

---

## 📈 Dashboard Features

### 1. Employee Overview

* Active vs Non-active employees
* Gender distribution
* Location filters (Houston, LA, NYC, Seattle)

### 2. Salary Analysis

* Salary comparison across groups
* Full-time vs Part-time segmentation

### 3. Workforce Trends

* Hiring trends over time (2013–2018)
* Headcount by department

### 4. Education Insights

* Employee distribution by education level:

  * Bachelor
  * Master
  * Doctor
  * College

---

## 🔍 Key Insights

* 📌 Salary varies significantly across employee groups, with Group D showing the highest range.
* 📌 Hiring dropped sharply in 2017 but rebounded strongly in 2018.
* 📌 Most employees hold Bachelor's or Master's degrees.
* 📌 Departments like **Tech Support and Research** have the highest headcounts.
* 📌 Workforce is evenly distributed across major locations.
